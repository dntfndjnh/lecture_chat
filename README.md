#  Lecture Q&A Chat Application (강의 질의 채팅 프로그램)

수업 중 질문하는 문화를 활성화하고, 교수님과 학생 간의 원활한 소통을 돕기 위해 개발된 **실시간 강의 질의응답 플랫폼** [2025 I&T 융합프로젝트]

---

##  개발 배경 (Problem Statement)
강의 중 궁금한 점이 생겨도 다음과 같은 이유로 질문을 망설이는 학생들의 'Pain Point'를 해결
* "질문이 흐름을 끊는 것은 아닐까?"
* "지금 하는 질문이 적절한 질문일까?"
* "질문 타이밍을 놓쳐서 수업이 끝나고 물어볼까?"

이러한 심리적 장벽을 제거하고, 즉각적인 소통 환경을 제공하여 학습 효율을 극대화하고자 기획함

##  주요 기능
* **실시간 채팅:** `Node.js`와 `Socket.io`를 활용하여 실시간 소통 환경 제공.
* **PDF 저장 기능:** 현재 대화 내용을 즉시 PDF로 변환 및 저장하여 복습 자료로 활용 가능.
* **로컬 로그 관리:** 모든 대화 내역이 KST 기준 로컬 파일(.txt)로 자동 기록되어 데이터 보존성 확보.
* **사용자 편의성:** .bat 실행 스크립트를 통한 간편한 서버 구동 및 직관적인 UI 설계.

##  기술 스택
* **Backend:** Node.js, Socket.io
* **Frontend:** HTML5, CSS3, JavaScript
* **Tools:** Git, Local File System API


##  기대 효과
1. **질문 문화 활성화:** 익명성 기반의 소통과 편리한 인터페이스로 질문의 부담 완화.
2. **학습 데이터화:** 저장된 질의응답 기록을 통해 강의 후 개인 복습 자료로 활용.
3. **강의 흐름 개선:** 교수님이 실시간으로 학생들의 궁금증을 파악하여 강의 방식에 즉각 반영.
