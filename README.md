강의 질의 채팅 프로그램 (Lecture Q&A Chat Application)
수업 중 질문하는 문화를 활성화하고, 교수님과 학생 간의 원활한 소통을 돕기 위해 개발된 실시간 질의응답 플랫폼입니다. 2025 I&T 융합프로젝트의 일환으로 제작되었습니다.


프로젝트 개요
수업 도중 즉각적으로 질문하고 싶지만, 흐름을 끊거나 적절한 질문인지 고민하게 되는 학생들의 '페인 포인트(Pain Point)'를 해결하기 위해 설계되었습니다. 실시간 채팅과 로컬 로그 관리 기능을 통해 질문의 부담을 줄이고 학습 효율을 높입니다.

주요 기능
실시간 채팅: Node.js와 Socket.io를 활용하여 지연 없는 실시간 소통 환경 제공.

PDF 저장: 대화 내용을 즉시 PDF 파일로 변환 및 저장하여 복습 자료로 활용.

로컬 파일 시스템 로그 관리: 모든 대화 내역이 서버 로컬에 자동 기록되어 데이터 보존성 확보.

사용자 편의 기능: 간편한 실행을 위한 .bat 파일 제공 및 직관적인 UI/UX 설계.

기술 스택
Backend: Node.js, Socket.io

Frontend: HTML/CSS/JavaScript (UI/UX 설계)

DevOps/Tooling: .bat 기반 자동화 스크립트, 로컬 파일 시스템 관리



 기대 효과
질문 문화 활성화: 익명성 또는 편리한 인터페이스를 통해 질문의 심리적 장벽 완화.

학습 효율 증대: 질문과 답변 내용이 자동으로 로컬에 저장되어 추후 개인적인 복습 데이터로 활용 가능.

강의 질 향상: 교수님은 학생들이 실시간으로 어떤 부분을 궁금해하는지 파악하여 강의 흐름에 즉각 반영 가능.
